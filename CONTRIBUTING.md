# 参与开发

本仓库只接受直接服务于 AI8video 的改动。

## 边界

- Web、CLI 与批量任务必须复用项目自有的 Python Runtime。
- 不引入通用 Agent 框架、Shell 工具、任意文件系统工具或通用网络工具。
- Python 是对话、任务状态、业务规则、媒体处理和持久化的唯一真值来源。
- 不重新引入无关的通用代理框架、聊天机器人前端、桌宠、TUI、记忆/反思模块或复制的第三方项目源码。
- 第三方依赖必须锁定版本并核对许可证；仓库不得提交 FFmpeg、模型、浏览器或其他大型二进制。

## 提交前检查

1. 运行与改动最相关的单元测试。
2. 运行 Python 编译检查、相关 Node 语法检查和 Shell 语法检查。
3. 模型与媒体测试必须处于 dry-run（试运行）或使用显式测试替身，不得消耗真实额度。
4. 媒体测试不得提交生成结果、用户素材、密钥或登录状态。
5. 新增依赖时同步更新 `THIRD_PARTY_NOTICES.md`。
